# 1학기 선택과목 교사 이동반 출석부

교사 이동반 출석부를 구글 스프레드시트를 DB로 사용해 관리하는 웹앱입니다.

## 배포 (Netlify)

1. 이 리포지터리를 GitHub에 올린 뒤 Netlify에서 연결합니다.
2. **Environment variables**에 `env.example`에 나열된 변수들을 설정합니다.
   - **필수**: `GOOGLE_SERVICE_ACCOUNT_JSON` — Google Cloud 서비스 계정 JSON 전체를 한 줄 문자열로 (줄바꿈은 `\n`으로).
   - **선택**: `SPREADSHEET_ID` — 기본 스프레드시트 ID (관리자 페이지에서 URL로 덮어쓸 수 있음).
   - **선택**: Firebase용 `VITE_FIREBASE_*` — 교사 이름 유지·구글 로그인 시 사용.
3. 빌드 후 배포합니다.

## 구글 스프레드시트 연동

1. [Google Cloud Console](https://console.cloud.google.com/)에서 프로젝트를 만들고 **Google Sheets API**를 사용 설정합니다.
2. **서비스 계정**을 생성한 뒤 JSON 키를 다운로드합니다.
3. 출석부용 **구글 스프레드시트**를 새로 만들고, 해당 시트를 **서비스 계정 이메일**에 편집 권한으로 공유합니다. (다른 교사의 출결 데이터가 노출되지 않도록, 앱이 서비스 계정으로만 시트에 접근합니다.)
4. 관리자 페이지에서 위 스프레드시트 주소를 입력하고, 이동반 출석부 엑셀·교사별 시간표 엑셀을 업로드한 뒤 **DB 제작 및 저장**을 실행합니다.
5. 교사들에게 공유하는 링크는 배포된 사이트 URL(예: `https://your-site.netlify.app/`)입니다.

## 사용 방법

- **교사**: 링크 접속 → 안내 확인 후 **시작하기** → 구글 스프레드시트 주소 입력·교사 이름 입력 → (선택) 구글 로그인 후 **시작하기** → 주간 시간표에서 과목 클릭 시 우측 출석 체크 패널에서 출결·비고 입력. 시간표 셀의 엑셀 아이콘으로 해당 시간 출석 xlsx 다운로드. **분반별 출석부 다운로드**에서 기간·분반 선택 후 PDF/CSV로 다운로드.
- **관리자**: `/admin` 접속 후 스프레드시트 주소·이동반 출석부 엑셀·교사별 시간표 엑셀 업로드 → DB 제작 및 저장 → 공유 링크 복사해 교사에게 전달.

## 엑셀 규격

- **이동반 출석부**: B1 과목명, B2 타임, B3 교실, B4 교사명(쉼표 구분), B5 인원, A6:E6 헤더(순번·학년·반·번호·수강생 이름), A7:E7~ 수강생 데이터. 공란 3행 후 다음 과목 블록 반복.
- **교사별 시간표**: 헤더 `teachername`, `dayindex`(월0~금4), `period`(1~7), `subject`(예: C고전문학감상), `room`.

## 로컬 실행

```bash
npm install
npm run dev
```

Netlify Functions는 로컬에서 `netlify dev`로 실행하면 됩니다.
